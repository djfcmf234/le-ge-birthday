## ? Features / 特性

- ?? **Bilingual** — Full Chinese & English support
- ?? **PWA** — Installable on mobile, works offline
- ?? **Time Gate** — Unlocks at 12:00 CST, countdown display
- ?? **High Scores** — Best records saved locally
- ?? **Glassmorphism UI** — Modern frosted glass design
- ?? **Confetti & Effects** — Celebratory animations

---

## ?? Tech Stack / 技术栈

- Pure HTML/CSS/JS — single file, no dependencies
- Service Worker for offline support
- Web App Manifest for PWA install
- CSS Grid & Flexbox layout
- Canvas for game rendering
